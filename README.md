## Cursed Edition Overview
<img width="800" height="420" alt="image" src="https://github.com/user-attachments/assets/6e9f305e-491c-406a-a0bd-fb0106714304" />

It adds exclusive features such as DFC rework, RBMK-like PWRs, the LFTR, and countless smaller features such as nuclear explosion VFX and SFX, more control panel support, rad classification, and more.


### Notes

The addon lacks an in-game version number system, so make sure you check on new releases every time you play!
